# The Beyoncé Effect

## Business Problem

Music streaming platforms rely on understanding listener behavior to improve engagement and recommend content.

This project analyzes how Beyoncé's albums and career events influence streaming trends using SQL, Python, and Power BI.


## Business Questions

1. Which album generated the highest popularity?

2. Which songs remain popular years after release?

3. Which audio features are most common among top-performing songs?

4. How do releases compare across albums?
